Introduction

This documentation provides a step-by-step guide to installing and configuring Node.js. Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside of a browser. It is widely used for building server-side and networking applications.

To install Node.js, you'll need basic knowledge of terminal commands like cd and mkdir.

Requirements

Before you begin the installation, ensure your system meets the following requirements:

  • Operating System: Windows 10 or later, macOS 10.14 or later, or Linux
  • Minimum 4 GB RAM
  • 500 MB of free disk space
  • Internet connection

To check your system version, run uname -a on Linux/macOS or winver on Windows.

Download

Download the Node.js installer from the official website:

https://nodejs.org

Choose the LTS (Long-Term Support) version for better stability. After downloading, verify the checksum of the file using sha256sum on Linux/macOS or CertUtil on Windows.

Installation

Follow these steps to install Node.js:

  1. Open the downloaded installer file.
  2. Follow the on-screen instructions to complete the installation.
  3. Once installed, verify by typing node -v and npm -v in your terminal to check that both Node.js and npm were installed successfully.

Configuration

After installation, configure Node.js by following these steps:

  1. Open your terminal.
  2. Set a global npm package directory with the command: npm config set prefix ~/npm-global.
  3. Add the directory to your PATH by adding export PATH=~/npm-global/bin:$PATH to your shell profile.

Troubleshooting

If you encounter issues during installation or configuration, refer to the following troubleshooting tips:

  • Ensure your system meets all requirements.
  • Make sure Node.js and npm are added to your PATH.
  • For issues with permissions, run commands as administrator (Windows) or use sudo on Linux/macOS.
  • Check the official Node.js documentation or community forums for more help.

For any other issues, try reinstalling or updating Node.js to the latest version.
